# Vinicius Nascimento – Personal Portfolio Website

## Project Description
This project is a **personal portfolio website** created to present my background, education, skills, and projects as an **incoming Electrical Engineering student at the New Jersey Institute of Technology (NJIT)**.  
The site was developed using **HTML, CSS, and JavaScript**, focusing on **responsive design**, **clean structure**, and **modern accessibility standards**.

The portfolio serves as both a professional introduction and a practice project for web development and deployment workflows.

---

## Technologies Used
- HTML5 – Semantic and accessible structure  
- CSS3 – Responsive and consistent styling  
- JavaScript (ES6) – Interactive and dynamic functionality  
- Git & GitHub – Version control and repository management  
- Vercel – Hosting and deployment  

---

## Features
- Responsive layout for all screen sizes (desktop, tablet, and mobile)  
- Clean and consistent design using external CSS  
- Semantic HTML elements for accessibility  
- Hamburger menu for mobile navigation  
- Image gallery or project carousel  
- JavaScript form validation on the contact page  
- Alt text for all images and accessible color contrast  

---

## Website Pages
- **Home** – A short self-introduction and navigation links  
- **About Me** – Education, background, and personal interests  
- **Projects** – At least three project showcases with images, descriptions, and links  
- **Skills** – Technical proficiencies (HTML, CSS, JS, Microsoft Office, etc.)  
- **Contact** – Contact form and personal contact details
